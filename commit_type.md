| Tür        | Ne zaman kullanılır?                    | Örnek                                     |
| ---------- | --------------------------------------- | ----------------------------------------- |
| `feat`     | Çalışan yeni özellik eklendiğinde       | `feat(uart): add line buffer`             |
| `fix`      | Bir hata düzeltildiğinde                | `fix(uart): rearm reception correctly`    |
| `docs`     | Yalnızca dokümantasyon değiştiğinde     | `docs: add UART debug notes`              |
| `test`     | Test veya test sonucu eklendiğinde      | `test(uart): add overflow recovery test`  |
| `refactor` | Davranış değişmeden kod düzenlendiğinde | `refactor(uart): extract transmit helper` |
| `chore`    | Bakım ve yardımcı işler yapıldığında    | `chore: update gitignore`                 |
